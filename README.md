# CodeAlpha_AI-chatbot
# 🤖 AI Chatbot using Python

## 📌 Project Overview

This project is a simple rule-based AI chatbot developed in Python. The chatbot interacts with users by responding to predefined questions and greetings. It demonstrates the basics of conversational AI using conditional statements and loops without requiring any external libraries or APIs.

---

## ✨ Features

* 💬 Interactive command-line chatbot
* 👋 Responds to greetings like **Hello**, **Good Morning**, and **Good Evening**
* 🤖 Answers basic questions about itself
* 😂 Tells a programming joke
* 🙏 Responds politely to "Thank You"
* 🚪 Ends the conversation when the user types **bye**
* ❓ Handles unknown inputs with a default response

---

## 🛠️ Technologies Used

* **Python 3**
* Conditional Statements (`if-elif-else`)
* Loops (`while`)
* Functions

## ▶️ How to Run

1. Install **Python 3** on your system.
2. Download or clone this repository.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python AI_chatbot.py
```
